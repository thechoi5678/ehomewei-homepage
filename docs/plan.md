# 홈페이지 제작 작업 계획

## 실행 순서

### Step 1. 지침 숙지
- CLAUDE.md를 읽고 역할과 금지 규칙을 확인한다
- docs/brand_voice.md를 읽고 디자인 방향과 기술 스택을 숙지한다

### Step 2. 브랜드 정보 수집
- 사용자가 제공한 링크(쿠팡 판매자샵 등)를 분석한다
- 아래 항목을 추출한다:
  1. 브랜드명 / 스토어명
  2. 판매 중인 주요 제품군
  3. 대표 제품의 핵심 특징
  4. 고객에게 강조할 신뢰 요소
  5. 차별화 포인트
- 확인된 정보와 추정 정보를 반드시 구분해서 표시한다
- 접근이 제한된 경우 추가로 필요한 자료 목록을 안내한다

### Step 3. 콘텐츠 초안 작성
- 섹션별 문구를 작성한다:
  - 메인 히어로 문구
  - 브랜드 소개 문구
  - 제품 소개 문구
  - 신뢰도 강조 문구
  - 외부 구매처 안내 문구
- 사용자 확인 후 다음 단계로 진행한다

### Step 4. 사이트 제작
- 전체 폴더 구조 생성
- 각 페이지 HTML/CSS/JS 코드 작성
- 모바일 최적화 적용
- SEO 기본 태그 적용

### Step 5. 배포 안내
- Vercel 배포 방법을 단계별로 안내한다
- 보유 도메인 연결 방법 (DNS 설정 포함) 안내
- 실제 접속까지 걸리는 시간 및 주의사항 안내

### Step 6. 결과물 확인
- 바로 실행 가능한 코드인지 확인
- 금지 문구(구매하기 등)가 없는지 최종 검수
